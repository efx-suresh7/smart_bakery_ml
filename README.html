Smart Bakery Management System

Local Environment Setup Guide for Beginners

Prerequisites

1. Git Installation

Used to clone and download the code repository.

  • Download the installer from git-scm.com/downloads (select the Windows option).
  • Run the file and proceed through all installation windows with default parameters.
  • Open your Command Prompt and type git --version to verify successful setup.

2. Python Installation

Required to train machine learning models and run the backend script.

  • Download the installer from python.org/downloads.
  • Launch the installer executable.
CRITICAL WARNING: Ensure that the checkbox labeled "Add python.exe to PATH" at the bottom of the installation setup window is checked. If omitted, python commands will not execute.
  • Click "Install Now" and wait for completion.
  • Open a new Command Prompt window, type python --version, and verify it outputs the version number.

Installation Steps

1. Clone Project Repository

Open your Command Prompt and clone the code directory to your Desktop:

cd Desktop
git clone https://github.com/efx-suresh7/smart_bakery_ml.git
cd smart_bakery_ml

2. Configure Python Virtual Environment

Isolates dependencies from global computer packages:

python -m venv .venv

Activate the environment:

  • Windows: .venv\Scripts\activate
  • macOS / Linux: source .venv/bin/activate

3. Install Python Dependencies

Downloads packages listed in the requirements configuration:

pip install -r requirements.txt

Running the Application

Launch the Flask backend server:

python app.py

Open your browser and navigate to the address below to access the dashboard interface:

http://127.0.0.1:5000
